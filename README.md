# dayone 2-to-1

Converts DayOne2 journal.json to DayOne Classic xml


# Installation

If you don't use `pipsi`, you're missing out.
Here are [installation instructions](https://github.com/mitsuhiko/pipsi#readme).

Simply run:

    $ pipsi install .


# Usage

To use it:

    $ dayone-2-to-1 --help
